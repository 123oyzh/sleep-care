# 智能睡眠环境调控设备 — MVP 功能清单

> 基于《智能睡眠环境调控设备-软件系统功能需求(实训需求)》V1.0 提取，聚焦实训可交付的最小可行产品范围。
> 本阶段不依赖真实硬件，所有设备数据与调控指令均使用模拟数据。

---

## 1. 用户登录

提供微信小程序端的手机号+验证码注册、微信一键授权登录两种入口，后端基于 JWT 维护登录态（Token 自动刷新），支持获取/修改个人昵称、头像、性别、出生年份等基本信息。

> 对应需求文档 §3.1 用户账号体系。MVP 阶段暂不含账号注销功能。

## 2. 虚拟设备管理

用户可通过"手动输入 16 位序列号"或"一键添加演示设备"两种方式绑定虚拟设备（后台自动标记 `is_virtual=1`），设备列表展示昵称、型号和固件版本，支持编辑昵称、解绑设备，虚拟设备默认显示"在线"状态。

> 对应需求文档 §3.2 设备管理（抽象层）。MVP 阶段暂不含 OTA 固件升级完整流程。

## 3. 睡眠报告分期图表

后台模拟数据生成器每日凌晨为绑定虚拟设备的用户自动生成符合生理规律的睡眠数据。小程序端以折线图或堆叠条形图展示睡眠阶段曲线（深睡、浅睡、REM、清醒四期），同时展示睡眠评分、总时长、深睡比例等核心指标。

> 对应需求文档 §3.3 睡眠报告（基础版+完整版核心图表）、§4.2 睡眠数据模拟生成器。

## 4. 噪音曲线

基于后台模拟生成的环境噪音等级数据（每分钟 dB SPL 值），在小程序睡眠报告详情页绘制夜间噪音随时间变化的折线图，并标注模拟异常噪音事件（如"呼吸暂停可能"、"周期性肢体运动"标签，生成概率 5%）。

> 对应需求文档 §3.3 完整版报告 — 环境噪音等级曲线、异常事件列表。

## 5. 日周月趋势

小程序支持按日、周、月三个时间维度查询历史睡眠报告：日报展示单晚详细分期数据，周报/月报以趋势折线图展示睡眠评分、总时长、深睡比例的连续变化，帮助用户发现睡眠质量变化规律。

> 对应需求文档 §3.3 报告查询 — 按日/周/月查看历史报告与趋势图。

## 6. 个性化作息

用户可设定目标就寝时间（`HH:mm`）和目标起床时间（`HH:mm`），配置助眠音效偏好（白噪音/海浪/雨声/篝火）、夜间勿扰时段开关等。所有设置存储于云端 `user_settings` 表，作为模拟数据生成器的输入参数，后续硬件接入后可同步至设备端。

> 对应需求文档 §3.8 个性化设置模块 — §3.8.1 作息计划、§3.8.2 助眠偏好、§3.8.4 设备设置（勿扰模式）。

## 7. 医生授权基础

患者在小程序中输入医生手机号发起授权请求，系统生成授权记录（默认有效期 30 天，支持 7/15/30/60 天可选）。医生通过医生端 Web 页面登录后可查看已授权患者的脱敏睡眠报告摘要（匿名 ID + 睡眠评分 + 报告图表）。患者可随时撤销授权，到期自动失效。

> 对应需求文档 §3.5 医生授权管理、§4.5 医生端 Web 后台。MVP 阶段暂不含 e-CBTi 专区（睡眠日记/量表测评/睡眠限制打卡）的数据授权查看。

## MVP 范围边界说明

| 纳入 MVP | 暂不纳入（后续迭代） |
|-----------|----------------------|
| 用户注册/登录（手机号+微信授权） | 账号注销 |
| 虚拟设备绑定/解绑/列表展示 | 真实硬件蓝牙配网 |
| 睡眠报告分期图表 + 噪音曲线 | e-CBTi 睡眠训练专区（卫生教育/日记/测评/打卡） |
| 日周月历史报告趋势 | 报告分享（截图/PDF/CSV导出） |
| 个性化作息设置（就寝/起床/勿扰） | 环境控制面板（亮度/色温/音量滑块） |
| 医生授权基础（发起/撤销/医生查看脱敏报告） | OTA 固件升级（仅保留版本号展示入口） |
| 后台模拟数据生成器 | 真实传感器数据接入 |
